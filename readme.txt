何が出来るの？
起動すると、勝手に喋ります。
特定の部位を撫でたり出来ます。今の所は頭……上の方だけです。

ヘンリックの中央辺りをクリックするとメニューが開きます。
・ゴースト呼びの鐘を鳴らす
同時に呼び出しているゴーストに挨拶をします。挨拶以外をする相手は居ません。
・話がしたい
自由入力で二人と話が出来ます。それなりのボキャブラリーを備える……予定です。
・喋り頻度
喋る頻度を設定出来ます。


出来ないことは？
気の利いた機能は搭載されていません。ミニゲームとか、二重人格とか。


どうしてクレープとタコスなんですか？
ヘンリックの帽子を書いた時に、友人から「タコス」にしか見えないと言われたから。
クレープはガスコインの帽子が丸くてクレープに似ているかもしれないと思ったから。
今思うとそうでもない。


既知の不具合
・帽子が重なる
→お手数ですが、ドラッグして位置調整をしてください。
